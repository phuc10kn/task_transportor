# 00 - Context

`00-context/` mô tả bối cảnh nền để hiểu project đang tồn tại trong hệ sinh thái nào và đang bị ràng buộc bởi điều gì.

Layer này trả lời:

- project đang tồn tại trong bối cảnh nào;
- phạm vi nền nào đang được giả định;
- premise, language, ecosystem và environment nào cần biết trước khi đọc layer khác.

## Universal Boundary

Layer này giữ model context chung, không giữ:

- product truth của một project cụ thể;
- business workflow chi tiết;
- domain rule hay technical mechanism.
