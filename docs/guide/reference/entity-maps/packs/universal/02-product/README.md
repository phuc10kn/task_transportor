# 02 - Product

`02-product/` mô tả sản phẩm phải cung cấp gì để đáp ứng business needs.

Layer này trả lời:

- user/business needs nào được chuyển thành capability;
- behavior, delivery, specification và acceptance nào cần có;
- scope nào thuộc sản phẩm và scope nào không thuộc.

## Covered Universal Concerns

- `01-needs/`
- `02-capabilities/`
- `03-behavior/`
- `04-delivery/`
- `05-specification/`
- `06-acceptance/`

## Universal Boundary

Layer này giữ product concern chung, không giữ:

- scope Lite/MVP riêng của một project cụ thể;
- UI composition detail;
- implementation structure và runtime operation detail.
