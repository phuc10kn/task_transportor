## Common statuses

| Status | Meaning |
| --- | --- |
| `draft` | Đang soạn, chưa review. |
| `active` | Canonical, đang hiệu lực. |
| `deprecated` | Không dùng cho instance mới. |
| `superseded` | Đã thay bằng instance khác. |
| `retired` | Kết thúc vòng đời. |
| `closed` | Hoàn tất. |

## Note types

| Note | Khi dùng |
| --- | --- |
| `NOTE-OPEN` | Thiếu thông tin hoặc chưa chốt. |
| `NOTE-CANDIDATE` | Đề xuất chưa canonical. |
| `NOTE-CONFLICT` | Có mâu thuẫn giữa nguồn. |
| `NOTE-EVIDENCE` | Cần nguồn chứng minh. |
| `NOTE-DECISION` | Cần decision trước khi chốt. |
| `NOTE-THEORY` | Cần xem xét theory. |

Không tự tạo status mới nếu meta chưa định nghĩa.
