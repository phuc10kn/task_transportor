# Example: Candidate To Canonical

## Case

Một context note rời rạc chứa cả phần dùng được và phần chưa đủ chín.

## Cách xử lý

Phần dùng được:

- relation type vs valid triple;
- canonical direction;
- sparse graph;
- trace validation workflow.

Promote vào:

```text
docs/meta/02-relation-types/README.md
docs/meta/03-rules/README.md
docs/meta/04-conventions/validation-model.md
docs/AGENT_SKILLS/guides/reading-strategy.md
```

Phần chưa đủ chín:

- core/supporting cheat sheet theo từng layer;
- external references chưa verify lại;
- gap assessment cần review riêng.

Không đưa phần chưa đủ chín vào `docs/workbench` vì workbench hiện chưa được đi vào hoạt động. Dùng `NOTE-OPEN` trong file canonical gần nhất hoặc giữ ngoài docs cho tới khi có home đang hoạt động.

File rời rạc gốc bị xóa để tránh có thêm source phụ.
