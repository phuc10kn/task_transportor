# Business Use Cases

Folder này giữ các use case nghiệp vụ chính của hệ thống.

## Nội dung chính

1. [usecase-map.md](usecase-map.md) - bản đồ use case tổng thể.
2. [access-and-project-admin.md](access-and-project-admin.md) - use case về đăng nhập và quản lý project.
3. [ingest-from-backlog.md](ingest-from-backlog.md) - use case đưa dữ liệu từ Backlog vào CIS.
4. [issue-preparation.md](issue-preparation.md) - use case chuẩn bị issue trước outbound.
5. [publish-to-jira.md](publish-to-jira.md) - use case preview và publish sang Jira.
6. [monitor-and-recover.md](monitor-and-recover.md) - use case monitoring, retry và truy vết.

## Ghi chú

- Use case trả lời câu hỏi: actor muốn đạt điều gì.
- Workflow trả lời câu hỏi: actor và hệ thống đi qua những bước nào.
