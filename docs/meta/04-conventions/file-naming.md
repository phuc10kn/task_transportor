# File Naming

## Entity instance

```text
README.md   = canonical entry point (bắt buộc)
```

Supporting docs chỉ khi cần:

```text
views/
examples/
assets/
notes/
```

## Meta files

```text
Entity Type definition  → {entity-type}.md
Relation Type definition → {relation-name}.md
Rules per layer         → valid-triples.md
```

## Theory files

```text
README.md
agent.md
theory.md
governance.md
```

## Không dùng

```text
index.md thay README.md cho entity canonical
version suffix: README-v2.md
```
