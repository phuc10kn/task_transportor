# Theory File Structure

Mỗi theory nằm trong:

```text
docs/theories/<slug>/
```

Structure chuẩn:

```text
README.md
agent.md
theory.md
governance.md
```

## Reading Levels

| File | Khi đọc |
| --- | --- |
| `README.md` | Entry point, summary, stable IDs |
| `agent.md` | Agent checklist, boundaries, common violations |
| `theory.md` | Deep reasoning, conflict, challenge, refine |
| `governance.md` | Reference notes, challenge, decision history |

Guide liên quan:

```text
docs/guide/concepts/theory-and-decision-model.md
docs/guide/unit-structure/theory/README.md
docs/meta/00-schemas/theory-package.md
```
