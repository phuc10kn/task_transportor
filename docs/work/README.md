# Tài liệu work

Thư mục này là nguồn thiết kế cho Central Sync Hub/CIS của `task_transportor`.

Mô hình chung:

```text
System -> CIS -> System
```

## Tài liệu điều hướng

- [00-overview.md](00-overview.md) - tổng quan sản phẩm, mục tiêu và mô hình hệ thống.
- [00-document-map.md](00-document-map.md) - bản đồ tài liệu theo scope.
- [00-reading-order.md](00-reading-order.md) - thứ tự đọc theo mục tiêu.
- [00-design-status.md](00-design-status.md) - trạng thái thiết kế, nguồn ưu tiên và việc cần rà khi implement.
