# FileAsset

| Field | Value |
|-------|-------|

| **layer** | `05-architecture` |

## meaning

Tài sản dạng file được quản lý như một phần của state ứng dụng nhưng lưu ngoài row data chính.

## use when

Khi project có attachment, exported artifact hoặc local object storage path.
